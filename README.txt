# Wrap Colors Academy — Static Site (PT/EN/ES)

## How to use
1) Replace the placeholder Hotmart links inside:
- /pt/index.html
- /en/index.html
- /es/index.html

Search for:
- #HOTMART_LINK_AUTO
- #HOTMART_LINK_DECOR
- #HOTMART_LINK_PPF
- #HOTMART_LINK_FILM
- #HOTMART_LINK_COMBO

2) Replace support email in the footer (search for support@YOURDOMAIN.com or suporte@SEUDOMINIO.com).

## Deploy on GitHub Pages
- Upload all files to your repo root.
- Enable Pages: Settings → Pages → Deploy from branch.
- Use branch main and folder / (root).
